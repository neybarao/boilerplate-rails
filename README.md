#Jera Boilerplate
Um workflow otimizado para os trabalhos de web na Jera

##Requisitos
  - **Command Line Tools**
  - **Homebrew**
  - **GIT**
  - **MySQL**
  - **RVM**
  - **Ruby 2.1.0**
  - **[LiveReload]**
[LiveReload]:http://blog.55minutes.com/2013/01/lightning-fast-sass-reloading-in-rails-32/
